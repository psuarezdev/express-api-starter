# Starter pack for backend web application development with Expressjs, Sequelize and JWT

## Introduction

This starter pack combines Expressjs, Sequelize and JWT for efficient backend web application development. Follow the steps below to set up your project

## Prerequisites

**[Node version 18.x.x](https://nodejs.org/)**  

## Setting up the project

### Cloning the repository

```shell
git clone https://github.com/psuarezdev/express-api-starter.git
```

### Install packages (once inside the repository)

```shell
npm i -E
```

### Setup .env files

```js
JWT_SECRET=

DB_NAME=
DB_USER=
DB_PASSWORD=
DB_HOST=
DB_DIALECT=
DB_POOL_MAX=
DB_POOL_MIN=
DB_POOL_ACQUIRE=
DB_POOL_IDLE=
```

### Start the app (from the root directory)

```shell
npm run dev
```
